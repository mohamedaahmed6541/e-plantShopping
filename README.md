# Paradise Nursery - Plant Shopping Application

A fully functional e-commerce React application for a boutique plant nursery. This project demonstrates state management using Redux Toolkit, responsive UI design, and automated deployment via GitHub Pages.

## ‚ú® Features

* **Interactive Landing Page:** An elegant entry point with an "Enter Site" transition.
* **Product Listing:** A categorized display of plants (Air Purifying, Aromatic, etc.) with a "Sale" badge and custom styling.
* **Real-time Cart Management:** * Add items to the cart with instant UI feedback (buttons turn gray and show "Added to Cart").
    * Floating cart icon badge that displays the total quantity of items dynamically.
* **Detailed Cart View:** * Adjust item quantities (Increase/Decrease).
    * Remove items entirely from the cart.
    * Automatic calculation of subtotals and total order cost.
* **Redux State Management:** Uses `@reduxjs/toolkit` and `react-redux` to maintain a global state for a seamless shopping experience.
